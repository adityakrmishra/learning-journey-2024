## Overview

Welcome to my 2024 learning repository. I am Aditya Kr. Mishra, a student at Bihar Vikas Vidyalaya. After establishing a strong foundation in 2023, my focus for 2024 shifted towards advanced programming concepts, full-stack development, and artificial intelligence. 

This repository documents my daily progress, technical experiments, and project implementations throughout the year 2024.

---

## Core Focus Areas

### 1. Advanced JavaScript and Full-Stack Development
Having covered the basics previously, I expanded my expertise in JavaScript to a professional level.
- **Concepts Mastered:** Asynchronous programming, Event Loop architecture, Closures, Promises, and ES6+ features.
- **Backend Engineering:** Node.js, Express.js REST APIs, Middleware implementation, and Authentication.
- **Database:** MongoDB and Mongoose ODM integration.

### 2. Python Domination and Advanced Paradigms
Python remains my strongest language. I advanced from basic Data Structures to system-level Python architecture.
- **Advanced Python:** Metaclasses, Generators, Context Managers, and Multi-threading/Multiprocessing.
- **Optimization:** Memory profiling and Big-O efficiency refactoring.

### 3. Artificial Intelligence and Machine Learning
This year marked my entry into the field of Artificial Intelligence. I progressed from fundamental data processing to intermediate model training.
- **Data Science:** Pandas, NumPy, and Matplotlib.
- **Classical ML:** Implementations of Linear Regression, Decision Trees, and K-Means clustering using Scikit-Learn.
- **Deep Learning:** Basics of Neural Networks using PyTorch.
- **Fine-Tuning:** Understanding Transformer architectures and writing basic scripts for model fine-tuning.

### 4. SQL and Relational Databases
Understanding that data is the backbone of any application, I dedicated significant time to relational databases.
- **Queries:** Complex JOIN operations, Subqueries, and Aggregate functions.
- **Advanced SQL:** Window functions, CTEs (Common Table Expressions), and performance indexing.
